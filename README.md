## 开始使用

1. 安装

使用本框架前，请在 CSS 中开启 border-box

```css
* {
  box-sizing: border-box;
}
```

## 开发

1. 安装依赖

```bash
npm install
```

2. 启动 server

```bash
npm start
```

3. 单元测试(只测一次)

```bash
npm run test

# 运行一次，每次保存都会测试，但是 window 上有 BUG

npm run dev-test

```
